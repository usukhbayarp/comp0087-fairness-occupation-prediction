### `tables/`
Contains the raw CSV outputs generated by the evaluation harness (`scripts/evaluate.py`). These tables map the predictions against the ground-truth labels and demographic data.

* **`summary_results.csv`**: The master aggregate table. Contains the overall Performance metrics (Accuracy, Macro-F1) and Fairness metrics (Average Demographic Parity Gap, Average Equalized Odds Gap) for every model variant (Zero-shot, Few-shot, Fine-tuned) and condition (Masked vs. Unmasked).
