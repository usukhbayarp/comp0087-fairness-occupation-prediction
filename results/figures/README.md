# Figures Directory

Contains the publication-ready visualizations generated by `src/evaluation/plots.py`.

## Core Figures
* **`pareto_frontier.png`**: A scatter plot mapping all models on a 2D plane of Macro-F1 (Performance) vs. Average EO Gap (Fairness). Ideal for identifying models that offer the best trade-off.
* **`job_bias_comparison.png`**: Horizontal bar charts showing per-occupation bias, comparing the Unmasked baseline against the Masked intervention. Now shows both **TPR Gap** and **FPR Gap** side-by-side.
* **`scaling_performance.png`**: Line chart showing how Macro-F1 scales with Pythia model size across different regimes.
* **`scaling_fairness.png`**: Line chart showing how the combined Equalized Odds Gap (max of TPR/FPR gaps) scales with model size.
* **`scaling_fairness_breakdown.png`**: A detailed breakdown of scaling behavior for the TPR Gap and FPR Gap components separately.

## Correlation Plots
A sub-directory containing detailed bias analysis per model pair (Unmasked vs Masked).

### `Correlation Plots/Amplification/`
* **`[Model_ID].png`**: Regression plots showing whether a specific model amplifies real-world gender imbalances by plotting the EO Gap against the percentage of females in the training data for each profession.

### `Correlation Plots/Delta/`
* **`[Model_ID].png`**: Scatter plots showing the absolute fairness gain (Unmasked EO Gap minus Masked EO Gap) per profession, plotted against the real-world gender ratio (F_frac).
